## Project Highlights

### TaskHorizon
**Task & Productivity Management System**  
**Stack:** Django, Python, Tailwind CSS

A productivity platform designed to help users manage tasks, track progress, and maintain execution clarity.

**Key Features:**
- Task lifecycle management (create, prioritize, complete)
- Due dates, status tracking, and progress indicators
- Responsive, clean UI optimized for daily use
- Scalable backend architecture for analytics and reporting

### QuickCart
**E-commerce Platform**  
**Stack:** Django, Python, Tailwind CSS

A server-rendered e-commerce application optimized for performance and simplicity.

**Key Features:**
- Product catalog, cart, and checkout workflows
- Tailwind-based UI for fast iteration
- Extensible architecture for payments and inventory


### MakeupBy Rose
**Beauty Brand Website & Booking Platform**  
**Stack:** Flask (Backend), React (Frontend)

A modern brand website with a React frontend and Flask REST APIs.

**Key Features:**
- Flask-based API architecture
- Responsive React UI aligned with brand identity
- Service listings and booking/inquiry workflows
- Mobile-first design for social traffic

### WeDrop
**Delivery & Logistics Platform (MVP)**  
**Stack:** Flask (Backend), React (Frontend)

A logistics management system focused on order tracking and operational visibility.

**Key Features:**
- Order creation and tracking APIs
- React dashboard for operations
- Role-based access control (admin, operations)
- Designed for future integrations and scale

### Breath
**Subscription-Based Information Platform**  
**Stack:** Django, Python, Tailwind CSS

A minimalist platform providing time-limited paid access to curated content.

**Key Features:**
- Time-based access control and session expiration
- Payment-gated content access
- Clean, single-page-style UI
- Planned multi-gateway payments (including M-Pesa)

### MindVista
**All-in-One Productivity & Planning System**  
**Stack:** Django, Python, Tailwind CSS

A comprehensive productivity platform combining planning, execution, and reflection.

**Key Features:**
- Goals, milestones, tasks, habits, and focus sessions
- Full authentication system with 2FA
- Modular architecture and scalable data models
- Planned analytics, charts, and mind-mapping features

### EchoAlert
**Safety & Awareness Platform (System Design)**  
**Stack:** Django (Planned)

A safety platform designed around alert escalation and awareness workflows.

**Key Features:**
- Defined alert escalation logic
- System architecture and UI direction
- Brand identity and platform planning completed

### Stock Control & Sales Management System
**Inventory & Sales Platform**  
**Stack:** Python

A dual-mode (web + desktop) system for managing stock and sales operations.

**Key Features:**
- Stock receiving and inventory tracking
- Sales processing with receipts and invoices
- Reporting and analytics planning
- Role-based access control

### DevOps CI/CD & IaC Implementation
**Practical DevOps Project (Academic & Applied)**  
**Stack:** GitHub Actions, Docker, CI/CD, IaC

A hands-on DevOps project focused on the synergy between development and operations.

**Key Features:**
- CI/CD pipelines using GitHub Actions
- Docker-based container workflows
- Tool comparison for CI/CD and Infrastructure as Code
- Monitoring concepts integrated into pipelines

### FINISH
**Internal Delivery & Operations System**  
**Stack:** Discord, Process Engineering

An internal execution system built to manage client delivery and developer workflows.

**Key Features:**
- Client-isolated Discord communication model
- Developer onboarding SOPs and rules of engagement
- Structured pricing, sales, and delivery processes
- Validation-focused execution framework

## Getting Started

### Prerequisites

* Node.js (v18+ recommended)
* npm or yarn

### Installation

```bash
git clone https://github.com/bugemarvin/portfolio.git
cd portfolio
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
